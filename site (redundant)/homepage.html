A bit about the project:

Hey! In the context of my research project, I'm set to develop a Python 
program. This program will furnish users with coding examples, prompting 
them to complete the provided code. In the event of an error, the program 
will offer suggestions on where the mistake may have occurred. To source 
questions, a web crawler will be employed to gather content from various 
online platforms, utilizing machine learning to identify and extract 
programming queries. The second step involves categorizing these programming 
questions and eliminating duplicates. This process aims to enhance the 
organization and efficiency of the generated questions. Finally, the project 
will leverage open-source, large language models to aid in code generation 
and completion, further enhancing the program's capabilities.
